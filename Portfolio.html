Research Intern — NIT Puducherry (Jun 2024 — Aug 2024)

DDoS Attack Prediction & Prevention System

Offline research internship focused on early detection and automated mitigation of Distributed Denial of Service (DDoS) attacks using machine learning and statistical techniques.
  • Duration: 3 months (Jun — Aug 2024)
  • Data & Tools: Packet captures (pcap), NetFlow statistics, Python, scikit-learn, pandas, feature engineering, Wireshark
  • Approach: Built a pipeline to extract per-flow and per-host features (e.g., packet rates, SYN/ACK ratios, bytes/sec, distinct source IP counts); used statistical anomaly detection and supervised ML (Random Forest, XGBoost) to classify attack vs normal traffic.
  • Prevention: Implemented a rule-engine that converts high-confidence detections into network filters (e.g., iptables rules / blackhole routes) and automated alerts to network operators.
  • Results: Achieved ~92% prediction accuracy on test sets; reduced detection response time by ~30% through optimized feature extraction and a real-time scoring pipeline.
  • Contributions: feature engineering, model training & evaluation, prototype mitigation scripts, and research write-up with recommended deployment steps for campus/edge networks.
Python scikit-learn XGBoost pcap / Wireshark Network security
Internship & Project — IoT Real-time Vehicle Crash Detection & Emergency Alert

IoT-based Real-time Vehicle Crash Detection & Emergency Alert System

End-to-end IoT project that detects vehicle crashes via sensors and automatically notifies emergency contacts and services in real time.
  • Scope: Sensor-based crash detection (accelerometer/gyroscope), edge processing, GSM/GPS based emergency alerting, and cloud logging.
  • Hardware: Raspberry Pi / ESP32 (edge node), MPU6050 (accelerometer + gyroscope) or ADXL335, GPS module, GSM module (SIM800L) for SMS/voice alerts.
  • Software & Cloud: Python (edge scripts), MQTT (Mosquitto) for telemetry, Node-RED for orchestration, Firebase/Firestore or simple REST backend to log events, simple React dashboard for visualization.
  • Method: Implemented threshold + ML-lite on-device detection: sudden delta-acceleration and orientation change triggers a local verification window; if confirmed, send automated alert with GPS & vehicle status to emergency contacts and dashboard.
  • Outcomes: Proof-of-concept performed live tests; alerts delivered reliably within seconds; system design emphasizes low-power edge inference and fallback SMS alerts when internet is unavailable.
Raspberry Pi / ESP32 MPU6050 / ADXL MQTT GSM (SIM800L) Python Firebase
